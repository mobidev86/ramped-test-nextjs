## Git clone
git clone https://github.com/mobidev86/ramped-test-frontend.git
## Pre-requisites
1. Node v20.x
2. Define the ENV file in the root project. A sample is attached for reference (.env.example)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build
$ npm run build
```
